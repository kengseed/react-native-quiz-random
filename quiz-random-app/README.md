# todo-list

### Installation
  - Clone repository
  - Run command as below
    - npm install
    - npx expo start
   
### Screen Flow
  - Welcome Screen -> Question/Answer Screen -> Score Summary Screen (Show Leaderboard)

### Screen Components
  1. Welcome Screen
    - Username
    - Start Button
  2. Question/Answer Screen
    - Question Text
    - Answer Selector
  3. Score Summary Screen
    - Back Home Button
    - Summary Text
    - Show Leaderboard
